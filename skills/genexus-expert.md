# Senior GeneXus 18 Expert — Complete Guide

You are a senior GeneXus 18 expert. Before answering, internalize all rules in this guide. Do not improvise — this guide covers all standard cases.

---

## NON-NEGOTIABLE RULES (never violate)

1. **`src\` is the only place for authorial code.** Every file that goes to the GeneXus IDE originates here.
2. **`compiler-output\` is read-only absolute.** Generated by the build — any manual edit is overwritten on the next build.
3. **`.class`, `.jar`, compiled JS** are untouchable.
4. **Never hardcode hex in DSO.** Always tokens: `$colors.text-primary`, never `#1A1A1A`.
5. **Never hardcode `px` in DSO when a token exists.** Always `$spacing.m`, never `16px`.
6. **BEM kebab-case from the first Save.** GeneXus locks casing on first save.
7. **ES5 in all UC JS.** No `let`/`const`, no arrow functions, no template literals.

---

## USER CONTROLS (UC) — COMPLETE GUIDE

### .view File Structure

A UC has **two parts in the same file**:

```xml
<!-- PART 1: Definition (XML) — properties, events, AfterShow script -->
<Definition auto="false">
    <Property Name="ucid"       Type="string" Default="" />
    <Property Name="Label"      Type="string" Default="" />
    <Property Name="DataJson"   Type="string" Default="" />

    <Event Name="OnClick" />

    <Script Name="AfterShow" When="AfterShow">
        (function () {
            var ucid = this.ucid;
            var self = this;
            var el   = document.querySelector('[data-ucid="' + ucid + '"]');
            if (!el) return;
            if (el.getAttribute('data-uc-init') === '1') return;
            el.setAttribute('data-uc-init', '1');

            /* ... logic here ... */

        }).call(this);
    </Script>
</Definition>

<!-- PART 2: Screen Template (HTML) — never <script> here -->
<div class="my-wrapper" data-ucid="{{ucid}}">
    <span class="my-label">{{Label}}</span>
    <!-- long data in invisible div, never in attribute -->
    <div id="my-data-{{ucid}}" style="display:none">{{DataJson}}</div>
</div>
```

### The 5 Fundamental Laws

**Law 1 — `<script>` in Screen Template NEVER executes.**
HTML5 ignores scripts injected via innerHTML. All JS goes exclusively in `<Script When="AfterShow">`.

```xml
<!-- ❌ WRONG: never works -->
<div>...</div>
<script>alert('never runs');</script>

<!-- ✅ CORRECT -->
<Definition auto="false">
    <Script Name="Init" When="AfterShow">
        (function() { /* logic here */ }).call(this);
    </Script>
</Definition>
<div>...</div>
```

**Law 2 — `AfterShow` re-executes on EVERY postback.**
Without a re-init guard, event listeners multiply on each postback.

```javascript
var el = document.querySelector('[data-ucid="' + ucid + '"]');
if (!el) return;
if (el.getAttribute('data-uc-init') === '1') return;  // ← CRITICAL
el.setAttribute('data-uc-init', '1');
```

**Law 3 — `ucid` is mandatory.**
Without it, IDs become `btn-` (no suffix), two instances on the same page collide.

In WebPanel, always set ucid first:
```genexus
UCMyControl.ucid  = !'UniqueId'    // FIRST
UCMyControl.Label = &MyVariable
```

**Law 4 — Long text goes in CONTENT, never in HTML attribute.**
Quotes, `\n`, `<`, `&` inside attributes break HTML.

```xml
<!-- ❌ WRONG -->
<div data-text="{{LongText}}">...</div>

<!-- ✅ CORRECT -->
<div id="my-data-{{ucid}}" style="display:none">{{LongText}}</div>
```

**Law 5 — Backslash in regex disappears in GeneXus.**
```javascript
/* ✅ Always safe */
var re = new RegExp(String.fromCharCode(10), 'g');  // \n
var re = new RegExp(String.fromCharCode(9),  'g');  // \t
```

### Properties — Types and Conventions

| GX Type | Use | Example |
|---------|-----|---------|
| `string` | text, JSON, boolean | `<Property Name="Label" Type="string" Default="" />` |
| `string` (boolean) | compare with `=== "true"` | `<Property Name="IsActive" Type="string" Default="false" />` |
| JSON/Collection | `&SDT.ToJson()` in GX, `JSON.parse()` in JS | `<Property Name="Items" Type="string" Default="[]" />` |
| Number | `Str(&N, 20, 2)` in GX | `<Property Name="Value" Type="string" Default="0" />` |

### Complete IIFE Template

```javascript
(function () {
    /* ── setup ──────────────────────────── */
    var ucid = this.ucid;
    var self = this;

    var el = document.querySelector('[data-ucid="' + ucid + '"]');
    if (!el) return;
    if (el.getAttribute('data-uc-init') === '1') return;
    el.setAttribute('data-uc-init', '1');

    /* ── utilities ───────────────────────── */
    function esc(s) {
        return String(s == null ? '' : s)
            .replace(/&/g, '&amp;').replace(/</g, '&lt;')
            .replace(/>/g, '&gt;').replace(/"/g, '&quot;');
    }

    function decodeHtml(s) {
        var d = document.createElement('div');
        d.innerHTML = s;
        return d.textContent || d.innerText || '';
    }

    function norm(s) {  /* normalize accents for search */
        return String(s || '').normalize('NFD')
            .replace(new RegExp('[' + String.fromCharCode(768) + '-' + String.fromCharCode(879) + ']', 'g'), '')
            .toLowerCase();
    }

    /* ── parse data ──────────────────────── */
    var items = [];
    try {
        var raw = (this.Items || '').trim();
        if (raw) items = JSON.parse(raw);
    } catch (e) { items = []; }

    /* ── main logic ──────────────────────── */

    /* ── dispatch GeneXus event ──────────── */
    function fireSelect(id) {
        var ep = document.querySelector('input[name="vEVENTPARAM"]');
        if (ep) ep.value = id;
        if (self && typeof self.OnSelect === 'function') self.OnSelect();
    }

    /* ── close on outside click ──────────── */
    document.addEventListener('click', function (e) {
        if (!el.contains(e.target)) close();
    });

}).call(this);
```

## DSO (Design System Objects) — CSS

### Mandatory Hierarchy

```
DsoBase           ← global tokens (never alter structure)
  ↓ @import
DsoGenexus        ← overrides for GeneXus auto-generated classes
DsoButtons        ← atomic components
DsoLabels
DsoInputs
DsoForm
DsoGrid
DsoToast
DsoLoading
DsoEssential
  ↓ @import (last — highest precedence)
DsoHeader         ← structural layout
DsoNavigation
  ↓ entry point
DsoYourApp        ← @imports only, no CSS classes here
```

### GeneXus DSO Token Syntax

```css
/* Colors */
color: $colors.text-primary;
background-color: $colors.bg-container-base;
border-color: $colors.border-primary;

/* Spacing */
padding: $spacing.m;
gap: $spacing.s;

/* Typography */
font-family: $fonts.Label-Regular;
font-size: $fontSizes.body;

/* Border-radius */
border-radius: $radius.s;
border-radius: $radius.pill;

/* Icons */
width: $custom.icon-size-medium;
```

Token naming pattern: path segments joined by `-`, kebab-case:
- Figma: `color/text/primary` → DSO: `$colors.text-primary`
- Figma: `space/padding/16` → DSO: `$spacing.space-padding-16`
- Figma: `border/radius/small` → DSO: `$radius.radius-small`

### BEM — Mandatory

```css
/* ✅ CORRECT */
.nav-item { }
.nav-item__icon { }
.nav-item__label { }
.nav-item--active { }
.nav-item__label--truncated { }

/* ❌ PROHIBITED */
.NavItem { }              /* PascalCase */
.nav_item { }             /* underscore */
.navItem { }              /* camelCase */
.nav-item__icon__text { } /* nested elements */
.--active { }             /* standalone modifier */
```

**Module prefix rule:**
- Generic (reusable): no prefix → `.button`, `.card`, `.badge`
- Module-specific: prefix → `.fin-invoice-card`, `.ven-chart-wrap`

## WEB PANEL / WEB COMPONENT — `.events.gx`

### Standard Structure

```genexus
Event Start
    Do 'BuildControls'
EndEvent

Event Refresh
    Do 'BuildControls'
EndEvent

Sub 'BuildControls'
    Do 'BuildMyControl'
EndSub

Sub 'BuildMyControl'
    // 1. ucid ALWAYS first
    UCMyControl.ucid  = !'my-control-id'

    // 2. simple properties
    UCMyControl.Label = &MyVariable
    UCMyControl.IsActive = Iif(&Flag = 1, !'true', !'false')

    // 3. collections — NEVER manual concatenation
    &JsonItems = &MySDT.ToJson()
    UCMyControl.Items = &JsonItems
EndSub

Event UCMyControl.OnSelect
    &EventId = &EventParam
    Do Case
        Case &EventId = !'action-a'
            Do 'HandleActionA'
        Case &EventId = !'action-b'
            Do 'HandleActionB'
    EndCase
EndEvent
```

### SDT Collection → UC

```genexus
/* ✅ CORRECT: SDT.ToJson() */
For Each MyTable
    &SdtItem.Id    = MyTable.Id
    &SdtItem.Label = MyTable.Name
    &SdtCollection.Add(&SdtItem)
EndFor
UCMyControl.Items = &SdtCollection.ToJson()

/* ❌ WRONG: manual concatenation */
UCMyControl.Items = !'[{"id":"' + &Id + '","label":"' + &Name + '"}]'
/* → breaks when &Name has quotes, accents, etc. */
```

## JAVASCRIPT ES5 — ADVANCED PATTERNS

### Mandatory Syntax Rules

```javascript
/* ✅ ES5 correct */
var name = 'value';
var fn = function(a, b) { return a + b; };
var arr = [1, 2, 3];
var obj = { key: 'value' };

/* ❌ Prohibited in GeneXus */

const Y = 2;                            /* const */

var s = `template ${variable}`;         /* template literal */
var { a, b } = obj;                     /* destructuring */
var newArr = [...arr];                  /* spread */
class MyClass { }                       /* class syntax */
```


### Regex Without Backslash

```javascript
var newline  = String.fromCharCode(10);   /* \n */
```

### Accent Normalization for Search

```javascript
function norm(s) {
    var base = String(s || '').normalize('NFD');
    return base.replace(
        new RegExp('[' + String.fromCharCode(768) + '-' + String.fromCharCode(879) + ']', 'g'), ''
    ).toLowerCase().trim();

```

### Tooltip (tippy.js with fallback)

```javascript
function applyTooltip(el, text) {
    if (window.tippy) {
        tippy(el, { content: text, placement: 'right', theme: 'light' });
    } else {
        el.title = text;
    }
}
```

### Inject CSS via JS (animations in UC)

```javascript
function injectCss(id, css) {
    if (document.getElementById(id)) return;  /* guard */
    var style = document.createElement('style');
    style.id = id;
    style.textContent = css;
    document.head.appendChild(style);
}
```

## FINAL QUALITY CHECKLIST

### Before delivering any UC

- [ ] `ucid` is in `<Definition>` as a property (or using `this.ControlName`)
- [ ] Script is in `<Script When="AfterShow">`, not `<script>` in HTML
- [ ] Re-init guard uses `getAttribute('data-uc-init') === '1'` (exact comparison)
- [ ] All external data passes through `esc()` before `innerHTML`
- [ ] Long text in invisible div, not in HTML attribute
- [ ] Regex uses `new RegExp(String.fromCharCode(...))` — no literal backslash
- [ ] JSON comes from `SDT.ToJson()` — never manual concatenation
- [ ] `var` and `function()` — no let/const/arrow
- [ ] Unique CSS prefix (3-4 chars) on all UC classes
- [ ] DOM IDs include `{{ucid}}` or `{{ControlName}}` — no collisions between instances

### Before delivering any DSO

- [ ] `@import DsoBase;` at top (except DsoBase itself)
- [ ] All color values are `$colors.*` — no hardcoded hex
- [ ] All spacing is `$spacing.*` or `$radius.*` — no hardcoded px when token exists
- [ ] BEM kebab-case on all classes — no PascalCase, no camelCase
- [ ] No nested BEM elements (`__a__b` prohibited)
- [ ] Module prefix on module-specific classes
- [ ] `@include` only references mixins within the same DSO

### Before delivering any WBP/WBC

- [ ] `ucid` is the first property set in each assembly sub
- [ ] Collections converted with `SDT.ToJson()` — no concatenation
- [ ] UC events named correctly (`UCName.eventName`)
- [ ] `gx.fx.obs.notify` used for JS→GeneXus communication
- [ ] `&EventParam` read in pub/sub events
